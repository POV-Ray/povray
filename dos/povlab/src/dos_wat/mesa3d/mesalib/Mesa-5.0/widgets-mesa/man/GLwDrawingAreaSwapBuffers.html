NAME

GLwDrawingAreaSwapBuffers - swap buffers in a GLwDrawingArea.

SYNOPSIS

#include<GL/GLwDrawA.h> void GLwDrawingAreaSwapBuffers(w) Widget w;

DESCRIPTION

GLwDrawingAreaSwapBuffers
is a front end to glXSwapBuffers, however instead of passing in a Display and a Window, a Widget is passed in.
