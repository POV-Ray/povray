Cylinder Object

We may also define a cylinder like this:

  cylinder {

    <0, 1, 0>,     // Center of one end

    <1, 2, 3>,     // Center of other end

    0.5            // Radius

    open           // Remove end caps

    texture { T_Stone25 scale 4 }

  }
