Point Lights

The simplest kid of light is a point light. A point light source sends light of the specified color uniformly in all directions. The default light type is a point source. The <Location> and COLOR is all that is required. For example:

  light_source {

    <1000,1000,-1000>, rgb <1,0.75,0> //an orange light

  }
